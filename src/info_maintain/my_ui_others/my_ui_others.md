2014年末刚开始找UI相关工作，面试一家公司修改页面，一位没有露面的前端技术头目托人事之口质疑了 我bootstrap模板改的在线简历不是手写代码，
开发改模板怎么了，而且我面试的是纯设计又不是前端，在线网站是为了表现有编程背景。最后说考考我改个页面，我说其它作品已经可以表现，一个页面需要很长时间，并且近期忙于面试。
最后还是做了，做了一天发对方邮件没回复😠。而且对方不像骗稿，只是无知，哎。
